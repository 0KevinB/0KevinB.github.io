- **Links:** [Live Web](https://www.medihelp.lat/) | [Mobile App Build](https://expo.dev/accounts/0kevinb/projects/medihelp-mobile/builds/384dd669-54eb-4a36-8ed6-19a5c018e788) | [Private Repo](https://github.com/0KevinB/MediHelp)

- **Tech:** Next.js 15, TypeScript, Tailwind, React Native, Firebase.
